vultr 는 웹서버 브랜드 이름

filezillar는 ftp client, 웹서버와 로컬을 이어주는 윈도우 탐색기 개념

** vultr 서버 로그인시, 비밀번호 수정하는 경우 filezillar 설정에서 동일하게 비밀번호 변경 필요

** 서버3대모양 아이콘 클릭시 사이트 관리자 이동(비밀번호 변경시)

** 아래 화살표는 로그인 바로가기 기능

filezillar,(ftp client) 이용하는 이유, 내 로컬의 파일들을 웹서버에 업로드 및 다운로드 도구

vultr 웹서버에 접속하는 또 다른 방법은, vultr terminal 로 접속

terminal 은 명령어 실행 가능, 파이썬 프로그램 실행, 크론탭 조정등

프로젝트 업데이트시 작업 순서

ftp 로 업데이트 된 프로젝트 파일 업로드, 웹 앱 같은 경우 파일 많음, ftp 폴더에 있는 쓸모없는 파일 삭제 필요(텍스트 파일)

local 에만 dev_txt 폴더에 저장, ftp 에는 업로드 필요 없음

ftp 업로드(local to ftp)_git 폴더는 절대 업로드 금지

vultr 터미널 접속(비번 메모 중요)

해당 웹앱의 폴더로 들어감(폴더 경로)_local 폴더 경로와는 다른 의미

웹서버에 웹앱이 있는 폴더 경로로 이동

vultr 터미널 이전 실행 명령어 소환 단축키--> 위 방향키

cd /var/choice
