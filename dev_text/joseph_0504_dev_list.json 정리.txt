stock_list.json 과 동일


기존에 있던 리스트는 버리고 이걸로 대체해줘


stock_list.json : 수정 금지 파일

stock_list.json = 수정 금지 원본 데이터 파일
/api/stocks = 원본 데이터 파일을 읽어 자동완성용 목록으로 제공
검색 자동완성 = /api/stocks를 주기적으로/입력 시 갱신해서 사용

서버가 stock_list.json의 수정 시간 mtime을 같이 내려줌
프론트가 마지막 mtime과 다를 때만 목록 교체
자동완성 드롭다운 열릴 때마다 최신 여부 확인
필요하면 3~5초 간격으로 자동 갱신
