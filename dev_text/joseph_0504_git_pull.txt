최신 커밋으로 원격저장소 덮어쓰기,  최신 상태
git fetch origin  
git reset --hard origin/main

git fetch origin 상태를 가져오는데 파일까지 가져올려면 
git reset --hard origin/main 말고 다른명령어는 없는거야??

fetch = 원격 상태 정보만 가져옴, 파일 안 바뀜

pull = 원격 파일 변경사항까지 내 브랜치에 반영
git pull origin main
or 
git merge origin/main

reset --hard = 내 로컬을 원격 브랜치와 강제로 동일하게 맞춤

push --force = 원격 저장소를 내 로컬 커밋으로 강제로 덮어씀


원격 저장소 최신 상태로 내 로컬을 완전히 똑같이 맞추고 싶다
git fetch origin
git reset --hard origin/main
git clean -fd

*원격 저장소 최신 변경사항을 내 작업에 합침
git pull origin main = git fetch origin + git merge origin/main
git fetch origin + git reset --hard origin/main 와 똑같은거 같음*

내 로컬 커밋을 살리면서 원격 최신 위로 정리
(로컬 변경사항을 유지하면서 합침)
git pull --rebase origin main = git fetch origin + git rebase origin/main
