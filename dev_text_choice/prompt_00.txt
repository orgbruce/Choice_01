vs code 에 넣을 프롬프트야, 내용 확인후 보충할 부분 알려줘
======================

테이블 헤더의 “현재가” 오른쪽에 “평단가” 컬럼을 추가해줘.

평단가는 각 종목별로 내가 직접 입력할 수 있어야 해.
각 종목 행의 “평단가” 셀을 클릭하면 앱 내부 팝업창이 뜨고, 숫자를 입력한 뒤 확인을 누르면 해당 종목의 평단가가 저장되게 해줘.

입력/표시 규칙:
- 숫자만 입력 가능하게 해줘.
- 표시할 때는 천 단위 콤마를 넣어줘. 예: 1234567 → 1,234,567
- 빈 값으로 확인하면 평단가를 삭제하거나 “-”로 표시되게 해줘.
- 저장된 평단가는 새로고침 후에도 유지되게 해줘.

위치는 “가격” 컬럼 그룹에서 “현재가” 오른쪽에 배치해줘.
기존 종목 메모 기능, 탭 기능, 컬럼 표시/숨김 기능과 충돌하지 않게 구현해줘.
